# Search Song By Name

Tired of going to Newgrounds to find the ID of a song you want to use? Simply type in the name of the song you want to use where you normally type the ID, and the first search result will be selected as the song for your level.

Currently only checks for songs on Newgrounds, although I may update it to query the music library as well.

## Note:
This mod is only functional due to the deactivation of Newgrounds anti-bot systems. If Newgrounds chooses to reactivate such systems, this mod will no longer be functional.

## Build instructions
```sh
geode build
```
